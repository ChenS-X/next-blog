# Welcome to my blog!

This is my first post. I'm using **Next.js** for the frontend and **Tailwind CSS** for styling.

## Why Next.js?

Next.js is great for blogs because of its **Static Site Generation (SSG)** capabilities. It makes the site fast and SEO-friendly.

### Features:
- Fast loading
- SEO optimized
- Easy to deploy to GitHub Pages
- Markdown support

Enjoy reading!
